# Rift-Equipment-Upgrade-Simulator
A simulator for an auto chess game


-Rift Equipment Upgrade Simulator is an app that simulates the upgrade mechanic of King God Castle (KGC),
King God Castle is an autochess game developed by Awesomepiece.

This project is merely intended to be used as a tool and is unintented to commercially profit.
